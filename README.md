🎙️ SOLO – Web-Based Voice Assistant Interface
📌 Project Overview

SOLO is a web-based voice assistant interface inspired by AI assistants like Jarvis. It provides an interactive UI with voice effects, animations, and dynamic text rendering. The project is built using HTML, CSS, and JavaScript, and is designed to simulate a smart assistant experience directly in the browser.

This project demonstrates frontend development skills including UI design, DOM manipulation, animations, and audio integration.



🚀 Features

🎤 Voice assistant-style interface

🔊 Integrated sound effects

✨ Animated text effects

🎨 Clean and responsive UI

📜 Interactive JavaScript controller logic

📦 Organized asset management

🛠️ Technologies Used

HTML5 – Structure of the web application

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Logic and interactivity

jQuery – DOM manipulation support

Animate.css – CSS animations

Textillate.js – Animated text effects

⚙️ Installation & Setup
Option 1: Run Directly in Browser (Recommended)

Download or clone the project.

Navigate to the www folder.

Open index.html in your browser.

No additional installation is required for frontend execution.

Option 2: Using Live Server (Recommended for Development)

Open the project folder in VS Code.

Install the Live Server extension.

Right-click on index.html.

Select Open with Live Server.

🎧 How It Works

The interface loads animated text using Textillate.js.

Sound effects are triggered from the assets/audio folder.

controller.js manages assistant behavior and interactions.

main.js initializes and connects UI elements.

CSS animations enhance visual presentation.
